¿Qué es InventiX?

InventiX es una aplicación web diseñada para pequeños negocios que necesitan gestionar:

  • Clientes: Registra y administra información de tus clientes
  • Productos: Controla tu inventario en tiempo real
  • Ventas: Registra cada transacción de forma fácil
  • Reportes: Genera reportes en Excel y PDF
